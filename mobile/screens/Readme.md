# Screens Directory

This directory contains the screen components for the Laborator Emanuela mobile application:

1. `HomeScreen.js` - Displays the menu categories and product listings.
2. `DetailsScreen.js` - Detailed product description with options to add to cart.
3. `CartScreen.js` - Review items in the cart and proceed to checkout.
4. `LocationsScreen.js` - Interactive map and list of Laborator Emanuela locations.
